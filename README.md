 HEAD

Application simple permettant d’ajouter et de gérer des tâches.


Ce projet a pour but de créer une petite application web qui permet :
D’ajouter une tâche avec un titre et une description
D’afficher la liste des tâches
De gérer les tâches de manière intuitive

HTML: structure de la page
CSS: design et mise en forme
JavaScript : fonctionnalités dynamiques
